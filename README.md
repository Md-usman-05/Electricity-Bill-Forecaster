# Electricity Bill Forecaster

## Overview
Electricity Bill Forecaster is a Machine Learning project that predicts monthly electricity consumption and the estimated electricity bill using Multiple Linear Regression.

## Features
- Predict monthly electricity units
- Estimate electricity bill
- Interactive Flask web application
- Data visualization using EDA
- Model evaluation using MAE, RMSE and R² Score

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Flask

## Machine Learning Algorithm
Multiple Linear Regression
